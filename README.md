## scripts

```bash
# install
npm install

# develop
# http://localhost:9002/#/login
npm run dev

# 预览发布环境效果
npm run preview

# 代码格式检查

npm run lint

# 代码格式检查和自动修复
npm run lint --fix

# svg压缩
npm run svg
```
